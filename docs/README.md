## Overview

DTC is a web-based hosting control panel. It manages:

- Virtual hosts (Apache / Nginx)
- DNS zones
- Mail (Postfix + Courier/Dovecot)
- MySQL/MariaDB databases
- FTP accounts
- SSL certificates

For full narrative documentation see the upstream HTML docs or visit the project page on GitHub.
